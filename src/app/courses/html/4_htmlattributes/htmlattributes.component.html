HTML Attributes

HTML attributes provide additional information about HTML elements. They are placed within the element's opening tag and consist of two parts: the name and the value.

  • Name: Specifies the property for that element.
  • Value: Sets the value of that property for the element.

There are two ways to specify the URL in the src attribute:

1. Absolute URL - Links to an external image that is hosted on another website. Example: src="https://www.randomimg.com/images".
2. Relative URL - Links to an image that is hosted within the website. Here, the URL does not include the domain name and the images is also hosted along with the page.

TIPS:It is almost always best to use relative URLs. They will not break if you change domain.
